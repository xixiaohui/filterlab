# Image （不是 img 标签）

Image = 一份“可被处理的图像数据”

# Filter（不是按钮）

Filter = 一种“可重复应用的图像变换规则”

# Pipeline （这是关键）

Pipeline = Filter 的有序组合

决定执行顺序
管理多个 Filter
是“工程能力”的核心体现
Pipeline 让你从“玩滤镜”变成“设计图像处理系统”
